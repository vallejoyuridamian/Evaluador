
PARAMETROS VALIDOS PARA cmdopt y cmdsim
========================================

sala=archi_sala 
   Camino completo a la sala


[tmp_base=dir_tmp_base] 
   El valor por defecto en LINUX es /tmp/simsee_{$USUARIO} siendo {$USUARIO} el valor de la variable de entorno USUARIO.
   El valor por defecto en WINDOWS es \SimSEE\tmp_rundir

[ejecutor=idejecutor] 
   Nombre de la carpeta a crear en tmp_base para escribir archivos resultados de corrida
   Si se omite el parámetro se usa tmp_base como carpeta de resultados
 
[escenario=nombre_escenario] 
   Nombre del escenario a activar. Si se omite se usa '__principal__' que implica activar el escenario
marcado como "El Principal" en la Sala de Juegos.

[rama=nombre_rama] 
   En la optimización por árbol de escenarios identifica la Rama a Optmizar.

[capas=lista_capas]
   Permite pasar una lista de capas para crear y activar por parámetros un escenario "al vuelo".

[nhilos=n]
   Cantidad de hilos a FORZAR. Si n = 1 usa Optimización MONO_HILO sino intenta Forzar la creación de
n hilos. Si n = -1 se Fuerza la cantidad de Hilos a la cantidad de núcleos de la máquina.

[ntareas=n]
    Cantidad de tareas en que se fragmenta el trabajo. Si n = -1 se fragmenta en la misma cantidad 
que los hilos creados para ejecución multi-hilo.

[monitores=archi_monitores]
   Si se especifica un archivo de monitores se carga.

[macro=ordenes]
   Si se especifican ordenes a ejecutar por la sala una vez cargada se ejecutan.
   Las ordenes que sabe ejecutar una sala no se documentan aquí. Esto se utiliza
   para acciones como por Ej. Calibrar un cono de pronósticos.

[semilla=n]
   Indica que se quiere forzar una semilla aleatoria difernete de la cargada en la sala.
   Si se pasa la palabra randomize  se carga una semilla al azar basada sobre la cantidad
   de milisegundos de ejecución del sistema operativo.
 
    if interpreteDeParametros.valStr('NCronicasSim') <> '' then
      sala.globs.NcronicasSim:= interpreteDeParametros.valInt('NCronicasSim');


Ejemplo de llamada a cmdopt:

c:\simsee\bin\cmdopt sala=C:\SimSEE\corridas\uruguay_2014_2030\uruguy_2014_2030.ese nhilos=-1
